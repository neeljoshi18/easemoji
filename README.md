EaseEmoji is an iMessage keyboard extension that makes finding emojis faster. 

To use it, open the app, grant camera permission, and load it as an extension in iMessage. 

It is currently just an Xcode project, but if it gains enough traction I may publish it as a full app.

Load it via the small plus icon next to the text field. 

While composing a message, tap the plus, select EaseEmoji, and look at the camera. 

Smiling inserts a smiley face, laughing inserts a laughing face, and an angry expression inserts an angry face. 

Three options appear; tap the one you want based on your expression.

That’s it. Load the Xcode project, try it, and if you like it after the effort, please give me a star so I know people want it.
